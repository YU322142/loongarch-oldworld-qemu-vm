# LoongArch Old-World QEMU VM Launcher 20260608

This version contains open-source scripts and documentation for setting up a visible Windows-hosted LoongArch old-world ABI1.0 Linux/X11 desktop test environment.

Included:

- Visible QEMU desktop launcher script.
- Default networking, audio, and host shared folder.
- Loongnix Desktop mini qcow2 download and checksum script.
- Work-disk creation, snapshot, and reset scripts.
- Full usage guide for moving test packages into the VM and performing manual acceptance.
- Tested Loongnix mini `tty1` SSH enablement commands run as `root`: `systemctl enable ssh` and `systemctl start ssh`.
- Full KDE/Plasma or Xfce desktop installation/enabling workflow when Loongnix mini stops at `tty1`.
- .NET/Avalonia application test checklist.

Not included:

- QEMU binaries or EDK2 firmware.
- Loongnix system image.
- Writable VM work disk.
- User test software, runtimes, or logs.

Install QEMU and download the system image from the official Loongnix directory as described in README.
