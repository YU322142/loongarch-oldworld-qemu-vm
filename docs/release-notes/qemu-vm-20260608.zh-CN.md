# LoongArch 旧世界 QEMU VM 启动方案 20260608

本版本包含可开源的脚本和文档，用于搭建 Windows 主机上的 LoongArch 旧世界 ABI1.0 Linux/X11 可见桌面测试环境。

包含内容：

- 可见 QEMU 桌面启动脚本。
- 默认网络、声音和宿主机共享目录。
- Loongnix Desktop mini qcow2 下载与校验脚本。
- 工作盘生成、快照和重置脚本。
- .NET/Avalonia 应用测试清单。

不包含内容：

- QEMU 二进制和 EDK2 固件。
- Loongnix 系统镜像。
- 虚拟机工作盘。
- 用户测试软件、运行库或日志。

请按 README 安装 QEMU，并从 Loongnix 官方目录下载系统镜像。
