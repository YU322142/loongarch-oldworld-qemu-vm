# Scripts

| Script | Purpose |
| --- | --- |
| `Install-Qemu-Windows.ps1` | Install the tested Windows QEMU package with winget |
| `Download-LoongnixImage.ps1` | Download and verify the Loongnix qcow2 image, then create a work disk |
| `Start-Loongnix-Desktop.ps1` | Start the visible LoongArch old-world X11 VM |
| `Stop-Loongnix.ps1` | Stop matching QEMU processes |
| `Reset-WorkDisk.ps1` | Recreate the writable work disk from the clean base image |
| `Package-Release.ps1` | Package scripts and docs for Actions or Release |
